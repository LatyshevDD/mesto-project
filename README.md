# Проект: Место

Проект “Mesto”
Функционал: одностраничный сайт, посвященный теме путешествий. Сайт адаптивен для основных устройств. Реализовано взаимодействие с сервером. Информация из форм, проходит валидацию и отправляется на сервер. Интерактивные элементы обновляются после загрузки информации с сервера. Сайт выполнен в парадигме ООП.

Технологии: HTML, CSS, Адаптивная верстка, Flexbox, GRID, JavaScript, Fetch, ООП, Webpack.

Ссылка: https://latyshevdd.github.io/mesto-project/
